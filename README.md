# 🏥 MediCare - Professional Health Platform

A fully functional healthcare reminder web application built with HTML, CSS, JavaScript and Firebase.

## 🌐 Live Demo
[Click here to open MediCare](https://anishpotdar55.github.io/-medicare)

## ✨ Features

### 👤 Patient Features
- 📋 Dashboard with health summary
- 💊 Medicine reminders with countdown timer
- ➕ Add / Delete medicines with stock tracking
- 📅 Appointment scheduling with doctor info
- 🛒 Pharmacy with medicine ordering
- 👤 Personal profile with medical info

### 👨‍⚕️ Doctor Features
- 📊 Clinical dashboard with patient stats
- 👥 Patient list with health scores
- 📅 Appointment management
- 📋 Write & manage prescriptions
- 📈 Reports and analytics

## 🔥 Tech Stack
- **Frontend** — HTML, CSS, JavaScript
- **Backend** — Firebase (Google)
- **Database** — Firestore
- **Authentication** — Firebase Auth
- **Hosting** — GitHub Pages

## 🚀 How to Use
1. Open the live link above
2. Choose your role — **Patient** or **Doctor**
3. Sign up with your email
4. Start managing your health!
